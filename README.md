# Champion Partners Website (React + Vite + Tailwind)

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages (auto)
This repo includes `.github/workflows/deploy.yml`.

1) Push to GitHub (branch: `main`)
2) Repo → Settings → Pages
3) Build and deployment → Source: **GitHub Actions**
4) Push again (or run Actions → Deploy to GitHub Pages)

GitHub will show your live URL in Settings → Pages.
